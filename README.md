# Student CRUD App (React + JSON Server)

## Features
- Add Student
- View Students
- Update Student
- Delete Student

## Tech Stack
- React (Vite)
- Axios
- JSON Server

## Setup Instructions

1. Install dependencies
npm install

2. Start JSON Server
npx json-server --watch db.json --port 3000

3. Start React App
npm run dev

## API Endpoint
http://localhost:3000/students
